To add staff photos:
1. Save photo as: images/team/dr-ward.jpg (or .png)
2. Recommended size: 400x400px, square crop, face centered
3. In index.html find the team-card for that staff member
4. Replace the .team-photo-placeholder div with:
   <img src="images/team/dr-ward.jpg" alt="Dr. Ward" class="team-photo">
5. Push to GitHub — Netlify will auto-deploy
